## WebRTC Server

- Node.js server
- [Android client](https://github.com/mailiang1995/ClientWebRTC.git)

## Install

It requires [node.js](http://nodejs.org/download/)

* git clone https://github.com/mailiang1995/ServerWebRTC.git
* cd ProjectRTC/
* npm install
* node app.js

The server will run on port 3000.
